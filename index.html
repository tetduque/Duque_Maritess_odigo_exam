A Warm Welcome

-The first breakpoint is around 768px width of screen size. I initialize the class col-md-3 so that it will be a 4 column layout.
-When it reaches a 480px width break point, i choose to have two columns which is the col-sm-6 class then inside of that div I created a another row and initiate a col-sm-6 class, so it will also have a two column.
-When it reaches smaller breakpoint let say 320px it will be on its default 1 column layout.
